Posters

Promotional Posters

Oct. 2022-Mar. 2023

Created for Capstone project promoting accessibility on campus. Leftmost poster was adapted from a past PSA project (see below) to be more accessible for people with colorblindness and to advertise our survey about disability concerns on campus for user research. The rightmost poster was the official promo poster for our project, where I combined two teammates' posters into one improved design and made layout adjustments.

Created using Adobe Illustrator

Map Access Features Icons

April 2023

Icons created to represent different access features for use on the final site's map for Capstone campus accessibility project. Access features displayed on map are shown to the right.

Access features represented: bench, wheelchair inaccessible elevator, wheelchair accessible elevator, ramp, out of order wheelchair inaccessible elevator, out of order wheelchair accessible elevator, steps, stairs, alternate stairs (unused)

Created using Adobe Illustrator

Department Logo

March 2022

Submitted to the Digital Media Studies department's logo redesign contest. The department combines technical skills with creative interests, so the design communicates this by using binary code in an artistic way. The department is concerned with digital media on a global scale, represented as a globe, as well as technology both old a new, represented with the more retro green on black text for the binary code. The department frequently used the old logo on stickers, which the emblem logo design is well-suited for.

Created using Adobe Illustrator

Moodboards

March 2022

Moodboards used to present the team's visual design choices for et Terra, a digital plant companion app. The leftmost was the initial concept and the rightmost was updated with the group's chosen colors and fonts. The layouts were currated to communicate a laidback design for the final product.

Created using Figma
